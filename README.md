# newsSearch
社会新闻检索系统，信息检索导论大作业<br><br>


## 开发
```bash
    # 克隆项目
    git clone https://github.com/zhenzhencai/newsSearch.git

    # 安装依赖
    npm install
    //or # 建议不要用cnpm  安装有各种诡异的bug 可以通过如下操作解决npm速度慢的问题
    npm install --registry=https://registry.npm.taobao.org

    # 本地开发 开启服务
    npm run dev
```
